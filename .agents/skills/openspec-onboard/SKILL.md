Guided onboarding flow:
1. Preflight CLI check
2. Suggest a small scoped task from codebase
3. Demonstrate explore mode briefly
4. Create change container
5. Build artifacts (proposal/specs/design/tasks)
6. Implement tasks
7. Archive change
8. Recap command reference

Use EXPLAIN → DO → SHOW → PAUSE transitions for teaching.
