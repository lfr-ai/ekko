# OpenSpec Guide

Use this as the command and workflow reference for OpenSpec in this repo.

## Core commands
- `/opsx:propose`
- `/opsx:explore`
- `/opsx:apply`
- `/opsx:sync`
- `/opsx:archive`

## Expanded commands
- `/opsx:new`
- `/opsx:continue`
- `/opsx:ff`
- `/opsx:verify`
- `/opsx:bulk-archive`
- `/opsx:onboard`

## Artifact flow
`proposal -> specs -> design -> review -> tasks -> apply`

## Validation
- `openspec validate --all --strict`
- `openspec schema validate ekko-production`
